git 常用命令集

git 常用操作练习点击这里，该网站常用命令show solution，reset，levels
删除当前项目的git关联
find . -name ".git" | xargs rm -rf
初始化
git init
查看追踪文件状态
git status
取消文件追踪
git reset HEAD [文件路径]
添加文件追踪
git add [文件]
提交
git commit -m [本次更新的消息提示]
创建切换分支
git branch bugFix;git checkout bugFix
或者
git checkout -b bugFix
合并当前分支与目标
git merge [目标分支]

git rebase [目标分支]
